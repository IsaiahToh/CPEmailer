How to Use

  1. Upload an Excel file containing email addresses (one per row in the first column)
  2. Upload your promotional flyer as an image or create one using HTML
  3. Configure your email settings (sender info, subject, etc.)
  4. Click "Send Emails" to start your campaign
  5. Monitor progress in real-time

Email Configuration Required

This app uses Nodemailer for sending emails. To set up email sending:

  1. Copy the environment file: cp .env.example .env
  2. For Gmail: Enable 2FA and generate an App Password
  3. For other providers: Configure SMTP settings in the .env file
  4. Set environment variables:
    • EMAIL_SERVICE=gmail (or outlook, yahoo, custom)
    • EMAIL_USER=your_email@gmail.com
    • EMAIL_PASS=your_app_password
  5. Restart the server: npm start
  6. Test your configuration using the test email feature above
